Eslint-config-silence содержит в себе только логические правила, без проверки стилистики. В первую очередь Silence запрещает устаревшие конструкции языка, помогает избежать ошибок в вашем коде, рекомендует наилучшие практики и обнаруживает "запахи кода". При выборе правил авторы следовали принципу сужения выбора и "ускучнения" языка.
